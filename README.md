# Basic Project of EJS
In this project the EJS template engine is used to take normal input and show to the client side. 

To use EJS

// first install ejs:
`npm install ejs`

// inside the server
`app.set('view engine', 'ejs');`

// create index.ejs inside views folder
`res.render('index',{});`

To run this project

    npm install
    npm start
